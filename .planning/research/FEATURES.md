# Features Research - Wine Matching MVP

## Table Stakes (must-have)

### Matching and Discovery

- Initial questionnaire with few but high-signal questions
- Ranked wine suggestions based on profile
- Option to disable match filter and browse all nearby wines

### Browse and Filters

- Distance slider/toggle
- Filter by winery, vintage, alcohol level, wine color
- Optional useful filter: price range

### Wine Detail

- Wine name, image, key traits
- Map location(s) where it can be found
- Drink status action (drunk/not drunk) feeding preference weights

### Data and Backoffice

- Local DB with wine catalog and winery geo coordinates
- Add workshop endpoint for non-winery businesses (bar, restaurant, club)
- Manage pages for winery and generic business to attach wines from catalog

## Differentiators

- Verified badge if winery adds production notes + additives
- Additive cards with modal details from DB
- Kiosk mode with QR handoff to mobile list

## Anti-Features (do not build now)

- Full ecommerce flow and cart
- Social feed/reviews network
- Advanced recommendation AI infrastructure
